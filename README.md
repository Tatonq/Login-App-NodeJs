# NodeJs-LoginApp
```
1. npm i
2. npm start
3. Open On http://localhost:3030/
```

## Dependencies
```
1.bcryptjs
2.body-parser
3.dotenv
4.ejs
5.express
6.jsonwebtoken
7.mongoose
8.mongoose-unique-validator
```
## Install
```
npm i bcryptjs body-parser dotenv ejs express jsonwebtoken mongoose mongoose-unique-validator nodemon
```
